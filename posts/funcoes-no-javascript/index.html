Tudo sobre Funções no JavaScript Diferente de outras linguagens, as funções em JS não são apenas mais uma de seu conjunto de instruções, mas sim objetos assim como Arrays, Number, String e Date. E como todo objeto em JavaScript elas possuem propriedades e métodos, podem ser atribuídas a variáveis e serem passadas como argumentos para outras funções. Elas são uma parte fundamental no desenvolvimento em JavaScript por causa de sua intensa utilização dentro da liguagem para os mais diversos propósitos, desde a criação de objetos a construção de padrões.* Função Declarativa Uma função em uma instrução propria, começando com a palavra-chave function. Funções declarativas são analisadas uma vez, são estáticas e recebem um nome para acesso. Função Anônima Uma função criada usando um construtor. É analisada cada vez que é acessada e não recebe um nome específico. Função Literal ou de Expressão Uma função criada dentro de outra instrução como parte de uma expressão. É analisada uma vez, é estática e pode receber ou não um nome específico. Se receber nome, este é acessível apenas dentro da própria função. Funções JavaScript criam escopo *Padrões Javascript
