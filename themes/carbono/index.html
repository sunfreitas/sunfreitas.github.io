Estrutura do Carbono

HTML Responsivo

O tema utiliza o mesmo esquema de grids do bootstrap para se adaptar desde dispositivos móveis aos largos monitores de PC.

Criado para performance

O Carbono utiliza BEM (Block Element Model) no CSS tornando fácil alterar partes específicas sem dificuldades.

Web semântico

Disponibilizamos o Sass do Carbono para que você mesmo escolha a melhor nomenclatura para suas classes CSS.
